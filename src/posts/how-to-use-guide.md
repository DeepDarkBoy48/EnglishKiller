# 如何使用 SmashEnglish 提升英语水平

欢迎来到 SmashEnglish！这是一个由 AI 驱动的英语学习助手，旨在通过深度分析和智能反馈帮助你掌握英语。

本文将为你介绍网站的三大核心功能：**句法分析**、**智能词典**和**写作润色**。

## 1. 句法分析 (Syntax Analyzer)

这是 SmashEnglish 最核心的功能。当你遇到长难句看不懂时，它能帮你抽丝剥茧。

### 如何使用：

1. 点击顶部的 **"句法"** 标签。
2. 在输入框中粘贴或输入你想要分析的英语句子。
3. 点击 **"分析"** 按钮。

### 你将获得：

- **句子成分可视化**：清晰地看到主语、谓语、宾语、定语、状语等成分。
- **详细解释**：每个单词或短语的词性、在句中的作用以及详细含义。
- **句型结构**：识别句子是 "主谓宾" 还是 "主系表" 等结构。
- **时态分析**：判断句子的主要时态。

## 2. 智能词典 (Smart Dictionary)

不仅仅是查单词，更是深度学习单词的用法。

### 如何使用：

1. 点击顶部的 **"词典"** 标签。
2. 输入单词。
3. 查看结果。

### 特色功能：

- **语境例句**：AI 会生成多个贴近生活的例句。
- **词根词缀**：帮助你记忆单词来源。
- **搭配用法**：列出常见的固定搭配。

## 3. 写作润色 (Writing Polish)

让你的英语表达更地道、更专业。

### 如何使用：

1. 点击顶部的 **"写作"** 标签。
2. 输入你写好的英语段落。
3. AI 会提供修改建议，包括语法纠正、用词优化和句式升级。

## 常见问题

### Q: 需要付费吗？

A: 目前本站完全免费，但你需要配置自己的 Google Gemini API Key。

### Q: 如何获取 API Key？

1. 访问 [Google AI Studio](https://aistudio.google.com/)。
2. 登录你的 Google 账号。
3. 点击 "Get API key" 创建一个新的 Key。
4. 回到本站，点击右上角的 **"设置 Key"** 按钮，填入你的 Key 即可。
